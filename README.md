# Smart Pedestrian Crossing Controller using Allen-Bradley PLC

This project is a PLC-based smart pedestrian crossing controller developed using RSLogix 500 and RSLogix Emulate 500.

## Features

- Vehicle traffic light control
- Pedestrian push button request
- Pedestrian walk signal
- Buzzer indication
- Night mode with yellow blinking
- Fault mode with vehicle safety priority
- Start and stop control

## Software Used

- RSLogix Micro Starter Lite
- RSLogix Emulate 500
- RSLinx Classic

## Project Aim

The aim of this project is to design a smart pedestrian crossing system for a straight road where vehicles normally move, and pedestrians can safely request crossing.
